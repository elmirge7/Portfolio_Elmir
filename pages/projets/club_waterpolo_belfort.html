Site du club de waterpolo de Belfort

Le projet consistait à refaire le site du club de water-polo de Belfort. Nous étions par équipe de 3. Je suis le chef de projet et le responsable du pôle technique dans le projet. Cela m’a permis d’obtenir une plus grande aisance dans le développement tout en pouvant toucher au design et à la communication.

La maquette

La maquette a été réalisée pour répondre aux attentes du commanditaire qui souhaitait un site plutôt structuré et avec obligatoirement un formulaire de contact. L’harmonie colorée se base sur le ballon de water-polo. Et la typographie est adaptée à l’environnement sportif.

La communication

À travers le plan de communication, nous avons imaginé un moyen de récolter de nouveaux adhérents dans le club. Cela passera par la création d’un compte Instagram directement relié au compte Facebook déjà existant. Nous avons aussi pensé à une modification des flyers pour les redistribuer. Mais aussi de faire des partenariats avec des acteurs locaux.

Le référencement

Nous avons, à travers différents audits, analysé le site déjà existant sous plusieurs axes. Celui de la sémantique, la technique, la visibilité, … Nous avons aussi proposé une campagne de netlinking. Nous avons utilisé des logiciels comme Screaming Frog, Ahrefs ou encore Majestic SEO.

Intégration maquette

La maquette HTML / CSS / JS a été réalisée à partir de la première version de la maquette XD. Celle-ci a été réalisée grâce au framework Bootstrap. Elle est totalement responsive.

Intégration finale

Le site est intégré sur WordPress à l’aide du plugin Elementor. La maquette a subi des changements. J’ai tout de même utilisé des notions de HTML / CSS / JS pour pouvoir le personnaliser au maximum. Le site est totalement responsive.

Outils utilisés

  • HTML
  • CSS
    • Flexbox
    • Media Queries
  • Javascript
  • Adobe XD
  • Bootstrap
