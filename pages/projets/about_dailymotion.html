About Dailymotion

Réalisation en groupe d’un site vitrine responsive en HTML / CSS / JS qui présente l’histoire de Dailymotion.

Introduction

Dans le cadre de ce projet en groupe, nous devions recueillir les besoins d'un client pour concevoir une interface web en utilisant des langages tels que HTML, CSS et potentiellement JavaScript. En plus des aspects techniques, nous devions également prendre en compte les critères ergonomiques et esthétiques pour concevoir une interface conviviale et attractive pour l'utilisateur.

Développement Front-End

Ce projet m’a donc permis de développer ses compétences en matière de développement front-end, de prendre en compte les besoins du client et les contraintes liées à l'utilisation de l'interface web sur différents types d'appareils. Mais aussi à apprendre à travailler en équipe. Cela m'a permis de mettre en pratique mes connaissances en matière de développement front-end. Pour la réalisation de ce projet, nous avons choisi d'utiliser les langages HTML, CSS et JavaScript. Nous avons conçu un site adapté pour les mobiles et autres types d'écrans en utilisant les media queries.

Outils utilisés

  • HTML
  • CSS
    • Flexbox
    • Media Queries
  • Javascript
  • Github
